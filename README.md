# batch-table

## 说明
batch表格
* 提供数据文件，数据列以**#**分割
* 直接执行table.bat，会展示demo表格
* 表格对齐方式有[居左\居中\居右]
* 表格风格多种，具体看预览
* *后续会将对齐方式、风格作为参数传入*
